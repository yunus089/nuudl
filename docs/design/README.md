# NUUDL Design Guides

Diese Dateien sind die einzige aktive Design-Quelle fuer `Landing + Consumer`.

- `design-system.md`: Farben, Typografie, Spacing, Radius, States
- `ui-principles.md`: Fokus, Hierarchie, Ebenen, NSFW/Paywall-Regeln
- `feed-design.md`: Feed-Rhythmus, Post-Struktur, Density und Scanbarkeit
- `microcopy.md`: Ton, Labels, CTAs, Empty States, Fehlermeldungen
- `visual-directions.md`: Zielaesthetik und finale Richtung
- `implementation-guide.md`: Reihenfolge, Guardrails und QA-Kriterien fuer die Umsetzung

Ab hier gilt:

- Bei Unklarheit konservativ nach Guide entscheiden
- Keine neuen Farben, Patterns oder Sondervarianten ausserhalb dieser Guides
- Alten Style entfernen, nicht mischen
